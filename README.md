## Requirements
* Unix system with i3 window manager
* Python3.x+
* Python3.x+ Pip

## Installation Instructions
```bash
git clone https://github.com/The-Repo-Club/i3-swallow
cd i3-swallow
cp i3-swallow ~/.local/bin/swallow
```
