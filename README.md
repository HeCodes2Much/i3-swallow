## Requirements
* Unix system with i3 window manager
* Python3.x+
* Python3.x+ Pip

## Installation Instructions
```bash
$ git clone https://github.com/Uniminin/Swallow/
$ cd Swallow
$ python3 -m pip install -r requirements.txt
$ sudo make install
```
